ÍCONE NECESSÁRIO
================

Coloque um arquivo chamado "icon.ico" nesta pasta (build/).

Requisitos:
- Formato: .ico
- Tamanho recomendado: 256x256 ou 512x512 pixels
- Pode conter múltiplos tamanhos (16, 32, 48, 256)

Ferramentas para criar ícones:
- https://www.icoconverter.com/
- https://cloudconvert.com/png-to-ico
- https://converticon.com/

Se não colocar um ícone personalizado, o Electron usará o ícone padrão.
